# Order Management System

A modern, full-featured Order Management System built with Next.js 16, React 19, and TypeScript. This application provides a comprehensive solution for managing orders, inventory, customers, dealers, suppliers, and products.

## Features

- **Dashboard** - Overview of key metrics and statistics
- **Order Management** - Create, view, and manage orders
- **Inventory Tracking** - Monitor stock levels and inventory
- **Customer Management** - Manage customer information and relationships
- **Dealer Management** - Handle dealer accounts and transactions
- **Supplier Management** - Track suppliers and procurement
- **Product Catalog** - Maintain product listings with images
- **Authentication** - Secure login and registration system
- **Theme Support** - Light/dark mode with customizable themes
- **Responsive Design** - Mobile-friendly interface

## Tech Stack

- **Framework**: Next.js 16.1.6
- **UI Library**: React 19.2.4
- **Language**: TypeScript 5.7.3
- **Styling**: Tailwind CSS 4.2.0
- **UI Components**: Radix UI
- **Forms**: React Hook Form + Zod validation
- **Charts**: Recharts
- **Icons**: Lucide React
- **Notifications**: Sonner

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd OMS3
```

2. Install dependencies
```bash
pnpm install
# or
npm install
```

3. Run the development server
```bash
pnpm dev
# or
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Available Scripts

- `pnpm dev` - Start development server
- `pnpm build` - Build for production
- `pnpm start` - Start production server
- `pnpm lint` - Run ESLint
